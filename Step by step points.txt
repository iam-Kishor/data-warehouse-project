Medallion Architecture -> bronze, silver, gold
Created dwh database with three schemas -> bronze, silver, gold
batch load -> auto deletes existing database and load all again, so please do careful while data modification

BRONZE LAYER:

data ingestion
data completeness and schema check

Bronze Layer Load:

Used ddl queries to Create blank tables and adding if-drop query to drop and load tables
Used bulk insert query to load data in bulk which is fast approach than normal insert
Used TABLOCK in query to lock the table during the bulk insert for better performance
Used Truncate before bulk insert query to prevent duplicate data loading
Created Procedure to do it fast and easy in a click
Used Try & Catch inside procedure for error handling
Used timers to record time taken for loading
